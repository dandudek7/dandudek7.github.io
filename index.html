Dan's Portfolio

Projects:


Pacmen Factory

This is a simulation of a Pacmen "factory" where moving Pacmen can be generated at varying speeds. Pacmen will collide with the edges of the screen but not with other Pacmen.

Eyes Exercise

This project shows a pair of animated eyes that follow the cursor around the screen. This is a simple example of using document.onmousemove to make websites more interactive by engaging the user with effects or info as they browse the site.

Bus Tracker

Using Mapbox's open source data, this map shows a marker that moves along the bus route from MIT to Harvard University, pausing at each stop. This can be used to track a bus in real time, like the popular college app Doublemap does for campus buses.
